## Quick Start Checklist

### Before Submission

- [ ] Finalize app icon (1024x1024 PNG, no alpha, no rounded corners for Apple)
- [ ] Take 8 screenshots on iPhone 15 Pro Max simulator (1290x2796)
- [ ] Take 8 screenshots on Android emulator (1080x1920)
- [ ] Create screenshot frames using Figma or screenshots.pro
- [ ] Record 30-second preview video
- [ ] Create Google Play feature graphic (1024x500)
- [ ] Review description copy one final time
- [ ] Paste keywords into App Store Connect
- [ ] Fill out Apple privacy nutrition labels
- [ ] Fill out Google Play data safety section
- [ ] Set age rating (4+ / Everyone)
- [ ] Set category (Lifestyle / Education)
- [ ] Prepare localized listings (ar, ur, tr)

### After Launch (Week 1)

- [ ] Monitor keyword rankings via App Annie or Sensor Tower
- [ ] Track conversion rates in App Store Connect / Google Play Console
- [ ] Respond to all reviews within 24 hours
- [ ] Submit to Islamic app directories and review sites
- [ ] Share launch on social media with screenshot assets

### Ongoing Optimization

- [ ] A/B test screenshots quarterly
- [ ] Update keywords seasonally (Ramadan, exam season, Eid)
- [ ] Refresh promotional text for Islamic holidays
- [ ] Add user testimonials to description as reviews come in
- [ ] Update What's New with every version release
