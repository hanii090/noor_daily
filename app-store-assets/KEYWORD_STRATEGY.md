## Seasonal Keyword Calendar

Swap keywords and promotional text based on Islamic calendar and academic seasons:

| Month(s) | Event | Keyword Swaps | Promo Text Focus |
|-----------|-------|--------------|-----------------|
| Mar-Apr | Ramadan | Add: ramadan, fasting, iftar | "Your Ramadan companion" |
| Apr-May | Finals (Spring) | Keep: exam, student | "Finals season? We've got you" |
| Jun | Eid al-Fitr | Add: eid, celebration | "Eid Mubarak! Keep the momentum" |
| Jul-Aug | Dhul Hijjah / Hajj | Add: hajj, dhul hijjah | "10 blessed days" |
| Sep | Back to School | Keep: student, exam | "Start the semester right" |
| Nov-Dec | Finals (Fall) | Keep: exam, student | "Finals stress relief" |
| Jan-Feb | New Year | Add: goals, habits | "New year, new spiritual habits" |

**Apple:** Keywords require a new version submission to change. Plan quarterly keyword rotations aligned with version updates.

**Google Play:** Description can be updated anytime. Update the short description and top of full description for each seasonal moment.

## Apple Search Ads Strategy (If Budget Available)

### Priority Keywords to Bid On
1. **Brand defense:** "noor daily" (exact match, low bid)
2. **Category:** "quran app", "islamic app", "hadith app" (broad match)
3. **Student niche:** "muslim student", "exam dua", "quran for students" (exact match, will be cheap)
4. **Competitor conquesting:** "muslim pro", "quran majeed" (competitor names, higher CPA but high intent)

### Recommended Budget Split
- 40% — Category keywords (volume)
- 30% — Student niche (high conversion, low cost)
- 20% — Competitor conquesting
- 10% — Brand defense

## Keyword Performance Tracking

After launch, track weekly:
1. **Keyword rankings** — Use App Annie, Sensor Tower, or AppFollow
2. **Search visibility score** — Composite of all keyword rankings
3. **Conversion rate by keyword** — Which keywords lead to installs
4. **Impressions vs. installs** — Identify where screenshots/description need work

### Target Metrics (First 90 Days)
| Metric | Target |
|--------|--------|
| App Store keyword rankings (top 10) | 5+ keywords |
| App Store keyword rankings (top 50) | 15+ keywords |
| Search conversion rate | > 30% |
| Browse conversion rate | > 25% |
| Average rating | > 4.7 |
